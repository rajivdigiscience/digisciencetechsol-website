Data handling

Minimum access, practical evidence

Preferred evidence

  • Read-only cloud access.
  • Billing and inventory exports.
  • Architecture diagrams and policy/configuration exports.
  • Screenshots where direct access is not appropriate.

Avoid sharing

  • Production passwords or long-lived admin credentials.
  • Secrets, API keys, private keys, or tokens in chat or email.
  • Unredacted regulated data unless an explicit handling process is agreed.
